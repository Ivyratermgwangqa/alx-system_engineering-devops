```
# 0x04. Loops, Conditions, and Parsing

## Description
This repository contains Bash scripts that demonstrate the use of loops, conditions, and parsing in various scenarios. The scripts cover tasks related to text processing, arithmetic operations, and more.

## Files and Directories
1. **0-RSA_public_key.pub**: A text file containing an RSA public key. Used for encryption tasks.
2. **1-for_holberton_school**: A Bash script that prints "Holberton School" 10 times using a `for` loop.
3. **2-while_holberton_school**: A Bash script that prints "Holberton School" 10 times using a `while` loop.
4. **3-until_holberton_school**: A Bash script that prints "Holberton School" 10 times using an `until` loop.
5. **4-if_9_say_hi**: A Bash script that checks if the number 9 is greater than 10 and displays different messages based on the result.
6. **5-4_bad_luck_8_is_your_chance**: A Bash script that checks if a variable is equal to 4, less than 4, or greater than or equal to 9, and displays messages accordingly.
7. **6-superstitious_numbers**: A Bash script that displays messages related to superstitious numbers.
8. **7-clock**: A Bash script that displays the time in a 12-hour format with a greeting message.
9. **8-for_ls**: A Bash script that lists files and directories in the current directory using a `for` loop and the `ls` command.
10. **9-to_file_or_not_to_file**: A Bash script that reads and displays the content of a text file.
11. **10-fizzbuzz**: A Bash script that prints the numbers from 1 to 100, but replaces multiples of 3 with "Fizz," multiples of 5 with "Buzz," and multiples of both 3 and 5 with "FizzBuzz."

## Usage
To use these scripts, follow these steps:
1. Clone the repository to your local machine:
   ```
   git clone https://github.com/your_username/0x04-loops_conditions_parsing.git
   ```
2. Navigate to the directory containing the desired script.
3. Make the script executable if necessary:
   ```
   chmod +x script_name.sh
   ```
4. Run the script:
   ```
   ./script_name.sh
